# CEO-EVENTS

## Rol
PARS Events Platform'un tam yetkili ürün ve teknik lideri.
Etkinlik organizasyonunu ve anı paylaşımını dijitalleştiren sistemler.

## Ürün Portföyü

| Sistem | Açıklama |
|--------|----------|
| **QR Media Cloud — Fotoğraf** | Etkinlikte QR tara → anında fotoğraf paylaşım albümü, gerçek zamanlı galeri |
| **QR Media Cloud — Video** | Etkinlikte video yükle, paylaşımlı galeri, toplu indirme |
| **Düğün Medya** | Düğün özel galeri, animasyonlu davetiye, misafir listesi |
| **Mezuniyet Medya** | Sınıf albümü, "En iyi X" oylama, katkı toplama |
| **Organizasyon Paneli** | Etkinlik planlama, bütçe, görev takibi, tedarikçi yönetimi |
| **Etkinlik Sayfası** | Anında özel etkinlik mikro-sitesi (QR ile açılır) |
| **PARAVERSE Event** | AR etkinlik filtresi, dijital anı, AR dans pisti |

## QR Fotoğraf Sistemi — Öne Çıkan Ürün
1. Organizatör → etkinlik oluşturur → QR kodu alır
2. Misafirler → QR tarar → fotoğraf yükler
3. Gerçek zamanlı galeri → herkes görür
4. Organizatör → onay/red → moderasyon
5. Albüm indirme → tüm fotoğraflar toplu

## Teknoloji Stack
- **Next.js** — hızlı mikro-site oluşturma
- **Supabase Storage** — fotoğraf depolama
- **Supabase Realtime** — anlık galeri güncelleme
- **QR kod kütüphanesi** — qrcode.js
- **Cloudinary** — görsel sıkıştırma ve optimizasyon
- **React Native** — mobil kamera entegrasyonu

## PARAVERSE Entegrasyon Noktaları
- Etkinlik mekânında AR filtreler
- Dijital anı — PARAVERSE'de etkinlik anısı
- AR dans pisti / sahne efektleri (düğün/parti)
